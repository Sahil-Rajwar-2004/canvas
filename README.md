# Canvas

### bring out the ***Picasso*** within you

## Direction to use it
|commands|info|
|-------|----|
|***`Esc`*** |Exit|
|***`i`*** |Display Info|
|***`r`*** |Red|
|***`g`*** |Green|
|***`b`*** |Blue|
|***`e`*** |Eraser|
|***`c`*** |Clear All|
|***`s`*** |Save File|
|***`+`*** |Increase Pen Size|
|***`-`*** |Decrease Pen Size|
|***`p`*** |Customize Color|
